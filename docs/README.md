# SyllabiQ (frontend) docs

This folder contains simple component and flow diagrams, plus the frontend data shapes used by the UI.

Files:
- `ARCHITECTURE.md` — short architecture overview
- `SCHEMA.md` — main request/response types
- `ER_DIAGRAM.mmd` — simple flowchart of frontend → backend
- `SEQUENCE_DIAGRAMS.mmd` — user query flow

How to view:
- In VSCode use a Mermaid preview extension or open `.mmd` files in `https://mermaid.live`.
